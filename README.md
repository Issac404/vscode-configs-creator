# VSCode Folder Creator

一个VSCode扩展，用于在工作区快速创建`.vscode`文件夹和自定义配置文件。

## 功能特性

- 🚀 **快速创建**: 在当前工作区一键创建`.vscode`文件夹
- 🎯 **多选支持**: 一次创建多个文件

## 使用方法

### 1. 创建.vscode文件夹

1. 打开VSCode工作区
2. 按 `Ctrl+Shift+P` (Windows/Linux) 或 `Cmd+Shift+P` (Mac) 打开命令面板
3. 输入 "cfgCreator..." 并执行
4. 会自动创建设置中预先填入的内容，如果需要自动创建其他的只需要在设置文件中修改
5. 如果文件已存在，会提示是否覆盖
